## Setup

```
git clone https://github.com/lapwat/secure-open-docker
cd secure-open-docker
```

Make sure you have docker-compose installed.
Put files to render into **data** folder.
Files ending in .url contains URLs

## Run

Run it!

```
./secure-open.sh
```
